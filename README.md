# 軽量キャプチャ (Lightweight Capture)

特定の領域をワンボタンで素早くスクリーンショット撮影するための、超軽量・ポータブルなWindows用キャプチャツールです。
枠だけが表示されるシンプルなUIで、背後のアプリの邪魔をせずに素早く撮影が可能です。

## 🌟 特徴

本ツールには、用途に合わせて使い分けられる **2つのバージョン** が同梱されています。

*   **`light_capture_COLORKEY.exe`（完全透過・クリックスルー版）**
    *   枠の内側が完全に透明になっており、**キャプチャ枠を重ねたまま背後のアプリ（ゲームやブラウザなど）を直接クリック・操作**できます。
    *   常に枠を画面上に置いておき、必要な瞬間に素早く撮影したい場合に最適です。
    *   ※ウィンドウが選択できなくなるなど動作が不安定な面があります。選択できなくなったらタスクバーからアプリを閉じてください。
*   **`light_capture_ALPHA.exe`（半透明版）**
    *   枠の内側がうっすらと色づいて（黒色）表示されます。
    *   背後のクリックはできませんが、「現在どこからどこまでの範囲を撮影しようとしているか」が視覚的に非常に分かりやすくなっています。
    *   透明版と比べて動作が非常に安定しています。

## 🚀 基本的な使い方

1.  `light_capture_COLORKEY.exe` または `light_capture_ALPHA.exe` を起動します。
2.  ウィンドウの端やタイトルバー（見えませんが掴めます）をドラッグして、キャプチャしたい領域に合わせます。
3.  **`Space` キー**を押すと、枠の内側が撮影され、PNG画像として保存されます。（撮影完了時に「ポン」と音が鳴ります）
4.  **`Esc` キー**を押すとアプリを終了します。

## ⚙️ 右クリックメニューの機能

アプリの枠上で **右クリック** するとメニューが表示され、以下の設定が可能です。

*   **保存先の設定**
    *   画像の保存先フォルダを指定します。（選んだフォルダは `savedir.txt` に記憶され、次回以降も引き継がれます）
    *   ※未設定の場合は、実行ファイルと同じ場所に自動的に `pictures` フォルダが作成され、そこに保存されます。
*   **設定 (位置・サイズの保存と呼び出し)**
    *   よく使うキャプチャの「位置とサイズ」に名前を付けて複数保存（プリセット登録）できます。
    *   **現在の枠の位置で登録**: 今ある枠の形をそのままワンポチで保存できます。
    *   **適用**: リストから選んで適用ボタンを押すと、即座にその位置へ枠が移動します。
    *   ※一番上の「前回終了時の位置」には、前回アプリを閉じた時の位置が常に自動保存されています。
*   **現在の位置・サイズの指定 (シンプル入力)**
    *   X, Y, 横幅(W), 高さ(H) の数値を直接手入力して、枠のサイズを微調整するためのシンプルなウィンドウです。

## 📁 構成ファイルについて

アプリと同じ階層に以下の設定ファイルが自動生成されます。これらをアプリと一緒に持ち運ぶことで、USBメモリ等に入れて別のPCでも同じ設定のまま使用できます（ポータブル運用可能）。

*   `config.txt`: 保存した位置・サイズのプリセット情報が記録されています。
*   `savedir.txt`: 画像の保存先フォルダパスが記録されています。（相対パスで `pictures` と記述することで、常にexeの横に保存させることも可能です）

## 🛠️ ビルド方法

MinGW (g++) などのコンパイラがインストールされた環境でビルド可能です。日本語（Shift-JIS）環境での文字化けを防ぐため、以下のコマンドを使用してください。

```cmd
# light_capture_COLORKEY.exe のコンパイル
g++ -O3 -static -static-libgcc -static-libstdc++ -fexec-charset=CP932 light_capture_COLORKEY.cpp -o light_capture_COLORKEY -lgdi32 -lcomdlg32 -lole32 -luuid -mwindows -lgdiplus

# light_capture_ALPHA.exe のコンパイル
g++ -O3 -static -static-libgcc -static-libstdc++ -fexec-charset=CP932 light_capture_ALPHA.cpp -o light_capture_ALPHA -lgdi32 -lcomdlg32 -lole32 -luuid -mwindows -lgdiplus
```
